Atividade Individual - DAC


Este projeto é uma simples modificação do código oferecido pelo professor para o desenvolvimento deste painel, como objetivo de avaliar de maneira individual a utilização do código disponibilizado, bem como a possibilidade de realizar seu deploy na web utilizando o Heroku, de forma que possa ser consultado posteriormente para avaliação da tarefa. O link para o github está disponível do classroom da disciplina.

Logo abaixo estão listadas todas as opções de acordo com o exemplo no qual o professor nos deu a base para a execução desta tarefa.
